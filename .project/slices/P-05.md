## Goal (validable)
1. **Real extraction.** `ShellExtractor().extract(path)` returns the file's
   function definitions as `Symbol`s â€” both `name() { â€¦ }` (POSIX) and
   `function name { â€¦ }` / `function name() { â€¦ }` (bash) forms. Each Symbol:
   `kind="function"`, `signature == f"{name}()"`, `is_public` by the leaf-name
   rule (`_is_public`), `docstring` = the contiguous `#` comment block
   immediately above the def (shebang excluded; `None` if none), `lineno`/
   `end_lineno` spanning the def via brace-depth matching, `arg_names=()`,
   `body_hash=None`. Deterministic & import-free (K0/K10) â€” never sources/execs
   the script.
2. **Registered by default.** At import, `register_extractor(ShellExtractor(),
   suffixes=(".sh", ".bash"))` runs, so `get_extractor("shell")` is a
   `ShellExtractor` and a `.sh`/`.bash` `symbols` ref with `lang: auto`
   resolves to it â€” with **zero** change to engine control flow (proves K0).
3. **End to end.** A `DocumentSpec` whose `symbols` ref targets a real `.sh`
   fixture (`lang: shell` AND `lang: auto`) builds a `DocumentSurface` over the
   shell functions; the eng-guide surface folds the leading-comment docstrings
   in, a user-guide surface drops `_`-prefixed helpers (K3) and excludes
   docstrings. A signature change (rename / new function) moves `surface_hash`;
   a comment-only edit OUTSIDE a leading doc-comment does not.

## Design
- **extract.py.** New `ShellExtractor` class (`language = "shell"`) +
  `_extract_shell_symbols(path: Path) -> list[Symbol]` (the body, mirroring
  `_extract_python_symbols`'s readâ†’ExtractionError contract, K8). Regex
  (module-level compiled, K10):
  - def header: `^\s*function\s+(NAME)\s*(?:\(\s*\))?\s*\{` OR
    `^\s*(NAME)\s*\(\s*\)\s*\{` where `NAME = [A-Za-z_][A-Za-z0-9_]*`.
  - `end_lineno`: scan from the opening `{`, counting `{`/`}` depth until it
    returns to 0 (best-effort; braces inside quotes/here-docs are a documented
    limitation â€” fall back to `lineno` if unbalanced).
  - docstring: walk UP from the header collecting contiguous `^\s*#` lines
    (stop at a blank/non-comment line; skip `#!` shebang), strip one leading
    `# `, join top-to-bottom; `None` if none.
  - `is_public = _is_public(name)` (shared leaf-name rule â€” consistency with K3).
  Module-level `register_extractor(ShellExtractor(), suffixes=(".sh", ".bash"))`
  after the class. Add `"ShellExtractor"` to `__all__`.
- **No engine edit.** `_symbol_language`/`_symbols_for_ref`/`build_document_surface`
  are UNTOUCHED â€” they already route by language via the registry (P-03). That
  untouched-ness IS the K0 proof.
