# Slice B-02 â€” heal never overwrites `human` regions (end-to-end)

EPIC B. Wires the `human` mode (B-01 schema) through drift + heal + monitor so a
human-owned region is **reported when stale but never auto-edited**. The
preservation guarantee lives at the WRITE boundary (heal), so it holds regardless
of which backend proposed the fix.

## Goal (validable)
A doc with a renderer-backed region (e.g. `symbols`) declared
`region_modes: {symbols: human}`: after a code change, `cdmon check` REPORTS the
region as drift but with `healable=False` (engine won't auto-edit), and
`cdmon monitor --apply` leaves that region's body **byte-identical** while still
refreshing the fingerprint / any non-human regions. Proven by a system test
asserting (a) the reported drift is healable=False, (b) the region bytes are
unchanged after `--apply`, (c) a non-human region in the same doc DID heal.

## Behavior to implement
- **`drift.detect`** â€” for a region whose `spec.mode_for(region_id)` is `HUMAN`:
  - if it has a known renderer and stored body â‰  rendered-expected â†’ emit a
    `Drift(kind=REGION, healable=False, detail="region '<id>' is human-owned
    (mode=human); engine will not auto-edit â€” review manually")`. (Reported, not
    auto-healed.)
  - suppress the `UNHEALABLE` drift for a human region with no known renderer
    (it's intentionally human, not an error).
- **`heal`** â€” the write boundary GUARANTEE. Thread an explicit
  `preserve: frozenset[str] = frozenset()` (region ids) through:
  - `_corrected` / `render_corrected` / `regenerate_regions`: never regenerate a
    region in `preserve` (skip it; leave its bytes).
  - `apply_fix(doc_path, fix, *, preserve=frozenset())`: when writing a whole-doc
    `new_doc_text`, **re-inject** the document's CURRENT bodies for every region
    in `preserve` (via `manifest.set_region`) before writing, so a backend that
    returned whole-doc text cannot clobber a human region. Region-shaped fixes
    targeting a `preserve` region are a no-op (return False).
  - Default empty `preserve` â‡’ byte-for-byte identical to today (additive).
- **`monitor.run`** â€” compute `preserve = frozenset(rid for rid in
  spec.region_keys if spec.mode_for(rid) is RegionMode.HUMAN)` and pass it to
  `apply_fix`. (Backend/FixRequest unchanged â€” the guarantee is enforced at
  heal, not delegated to the backend.)
