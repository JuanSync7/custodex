# Slice D-03/D-04 â€” similarity retrieval + agent few-shot exemplars

EPIC D. Mines the resolution substrate (D-01/02): retrieve the most-similar past
**resolved** drifts and feed them to the agent backend as few-shot exemplars, so
remediation improves from history. Builds on D-01/02.

## Goal (validable)
(D-03) `rank_similar(target, records, resolutions, top_n=3)` returns the N most
similar PAST RESOLVED records to a target drift/record, deterministically, with a
planted near-duplicate ranked first and unresolved records excluded. (D-04) when
exemplars are present, the agent backend's composed prompt INCLUDES them (a new
prompt artifact), proven offline with a fake driver; with none present the prompt
is byte-identical to today (additive). Proven by ranking tests + an agent
prompt-composition test.

## In scope
### D-03 â€” `code_doc_monitor/similar.py` (new, pure, deterministic)
```python
class Exemplar(BaseModel):              # frozen, extra="forbid"
    record: ReviewRecord
    resolution: ResolutionRecord
    score: float

def rank_similar(
    target: ReviewRecord,               # or a Drift â€” pick one, document it
    records: list[ReviewRecord],
    resolutions: list[ResolutionRecord],
    *, top_n: int = 3,
) -> list[Exemplar]: ...
```
- Population = records whose `record_id` is in `resolved_index(resolutions)`
  (RESOLVED only). Never the target itself.
- Score = deterministic weighted sum of feature matches: same `doc_id`, same
  `drift_kind`, same `audience`, same `region_id`, exact `surface_hash` match.
  Document the weights. Tie-break: higher score, then more-recent `resolved_at`,
  then `record_id` (total order â†’ deterministic, K10). Pure, no I/O.

### D-04 â€” agent few-shot exemplars (additive)
- `backends.FixRequest` gains `exemplars: tuple[Exemplar, ...] = ()` (additive;
  default empty â‡’ every existing backend test unchanged).
- `monitor.run`: OPT-IN retrieval (a `Monitor(use_exemplars=False)` flag +
  resolutions-log path, default OFF so existing behavior is byte-identical). When
  ON, before `backend.propose`, read the review log + resolutions, `rank_similar`,
  attach to the `FixRequest`. Keep it cheap; default OFF.
- `agent/` â€” a new prompt artifact `EXEMPLARS.md` (template/instruction) +
  `select_artifacts` includes it only when `req.exemplars` is non-empty +
  `render_context` renders the exemplars (drift â†’ resolution, esp.
  `resolved_text` for OVERRIDDEN). MockBackend IGNORES exemplars (stays
  deterministic). Offline fake-driver test asserts the exemplar text appears in
  the composed prompt when present, and is absent otherwise.
