# Slice A-03 — `coverage.py` ownership resolver

Part of EPIC A. The heart of the coverage system. Builds on A-01/A-02. Pure math:
cross the inventory (files + symbols) with `config.documents` code_refs to decide
what is **documented** vs an **undocumented gap**, at BOTH file and symbol
granularity (lossless).

## Goal (validable)

Given a `MonitorConfig` and a `SymbolInventory` over a real fixture mini-repo
where 2 of 3 files (and a subset of one file's symbols) are referenced by
documents, `resolve_coverage` reports **exactly** the unreferenced file and the
unreferenced public symbols as gaps, attributes each documented item to the
owning doc id(s), and computes correct file-level and public-symbol-level
percentages. Proven by a test asserting the exact baskets + percentages.

## In scope — `custodex/coverage.py` (new, pure; reuse `extract._select`)

Proposed contract (finalize + add to ARCHITECTURE.md before implementing):

```python
class OwnedFile(BaseModel):         # frozen, extra="forbid"
    path: str
    language: str
    owners: tuple[str, ...]         # doc ids whose code_refs name this path ((): gap)

class OwnedSymbol(BaseModel):       # frozen, extra="forbid"
    path: str
    name: str                       # qualified for methods (Class.method)
    kind: str
    is_public: bool
    owners: tuple[str, ...]         # doc ids whose code_refs select this symbol

class CoverageReport(BaseModel):    # frozen, extra="forbid"
    files: tuple[OwnedFile, ...]    # ALL files, sorted (lossless)
    symbols: tuple[OwnedSymbol, ...]# ALL symbols, sorted (lossless)

    @property
    def documented_files(self) -> tuple[OwnedFile, ...]      # owners non-empty
    @property
    def undocumented_files(self) -> tuple[OwnedFile, ...]    # owners empty
    @property
    def documented_symbols(self) -> tuple[OwnedSymbol, ...]  # public & owned
    @property
    def undocumented_symbols(self) -> tuple[OwnedSymbol, ...]# public & unowned (the gap basket)
    @property
    def percent_files(self) -> float        # documented / total files (100.0 if 0 files)
    @property
    def percent_public_symbols(self) -> float  # documented_public / total_public

def resolve_coverage(config: MonitorConfig, inv: SymbolInventory) -> CoverageReport: ...
```

### Ownership rules (precise)
- **File owned** iff some `document.code_refs[*].path == file.path`.
- **Symbol owned** iff some `code_ref` with `ref.path == file.path` *selects* it.
  Reuse `extract._select(symbols, ref.symbols, ref.lines, ref.names)` over that
  file's full symbol list and test membership — this gives the correct
  whole-file (empty selectors), `symbols` (incl. class→methods), `lines`-overlap,
  and `names` semantics for free. Do NOT re-implement selection.
- **Ignore audience for ownership.** A code_ref "covers" the symbol it points at
  regardless of audience; the audience filter governs the *hash/surface*, not
  whether a doc references the code. (Note this in LESSON_LEARNT — it's a
  deliberate divergence from `build_document_surface`, which DOES filter.)
- **Universe for the % gap metric = PUBLIC symbols.** Private symbols are tracked
  (lossless, in `symbols`) but are NOT counted in `percent_public_symbols` nor in
  the `undocumented_symbols` gap basket (they are not doc targets). `is_public`
  is already on each `Symbol`.
- Deterministic (K10): all tuples sorted (files by path; symbols by (path, name,
  kind)); owners sorted + deduped. Pure (K1), no FS mutation, no new dep (K0).

## Test plan (`tests/test_coverage.py`)

- **real fixture**: a `MonitorConfig` + mini-repo: file `a.py` referenced
  whole-file by doc D1; file `b.py` referenced by doc D2 with `symbols=[Foo]`
  (so `Foo` + `Foo.method` owned, a sibling `bar` unowned); file `c.py`
  referenced by no doc. Assert: `undocumented_files == (c.py,)`; `b.py`'s `bar`
  is an undocumented gap but `Foo`/`Foo.method` documented; D1 owns all of
  `a.py`'s symbols; percentages exact.
- **unit**: each selector kind (whole-file, symbols+method pull-in, lines
  overlap, names) attributes ownership correctly; a shared file referenced by two
  docs yields two owners (sorted); private symbol excluded from gap basket but
  present in `symbols`; empty repo → 100.0% both metrics.
- **determinism**: same inputs → identical report.
- Coverage of `coverage.py` ≥ 90% (aim 100%); whole-pkg gate ≥ 90%.
