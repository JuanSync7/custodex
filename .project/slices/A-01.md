# Slice A-01 — `inventory.py`: repo code-file discovery

Part of EPIC A (lossless coverage). First slice of the program. File-level only;
symbol-level is A-02.

## Goal (validable)

Given a fixture directory tree and include/exclude glob sets, `discover_files`
returns the **exact** expected set of repo-relative code-file paths, **sorted and
deterministic**, honoring excludes; a non-existent or non-directory root raises a
typed `CodeDocMonitorError` subclass. Proven by an automated test asserting the
returned tuple equals a hand-written expected tuple on a real fixture tree.

## In scope — `code_doc_monitor/inventory.py` (new, pure, K0/K1/K10)

Proposed contract (finalize + add to ARCHITECTURE.md before implementing):

```python
class CodeFile(BaseModel):          # frozen, extra="forbid"
    path: str                       # repo-relative, POSIX separators
    language: str                   # derived from extension ("python", ...)

class Inventory(BaseModel):         # frozen, extra="forbid"
    root: str
    files: tuple[CodeFile, ...]     # sorted by path, deduped

DEFAULT_INCLUDE: tuple[str, ...] = ("**/*.py",)
DEFAULT_EXCLUDE: tuple[str, ...] = ("**/.*/**", "**/__pycache__/**", "**/.venv/**")

def discover_files(
    root: Path,
    *,
    include: tuple[str, ...] = DEFAULT_INCLUDE,
    exclude: tuple[str, ...] = DEFAULT_EXCLUDE,
) -> Inventory: ...
```

- Pure + deterministic (K10): output sorted by `path`, no wall-clock, no FS
  mutation (K1). Paths POSIX-normalized and repo-relative to `root`.
- Matching: a file is included iff it matches ≥1 `include` glob and 0 `exclude`
  globs. Use `pathlib`/`fnmatch` (stdlib only — K0; no new deps).
- `language` mapping is a small extension table; unknown extension that still
  matched an include → `language="unknown"` (don't drop it — losslessness).
- Loud (K8): add `InventoryError(CodeDocMonitorError)` to `errors.py`; raise it
  when `root` does not exist or is not a directory, with a clear message.

## Test plan (`tests/test_inventory.py`)

- **unit**: glob include/exclude matching; default excludes drop `.venv`,
  `__pycache__`, dot-dirs; extension→language; unknown extension kept as
  `unknown`; determinism (same tree → identical tuple; order independent of FS
  iteration — build a tree whose natural os.walk order differs from sorted).
- **integration / real fixture**: build a realistic mini-repo under `tmp_path`
  (`pkg/a.py`, `pkg/sub/b.py`, `pkg/__pycache__/x.pyc`, `.venv/lib/y.py`,
  `tests/test_z.py`, a `README.md`) and assert the exact included set.
- **loud**: missing root and a file-as-root both raise `InventoryError`.
- Keep coverage of `inventory.py` ≥ 90% (aim 100%); whole-pkg gate ≥ 90%.
